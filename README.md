This database is modeled on a University Enrollment System. It contains eight interlinked datasets with sample records to support analysis and SQL operations.
The eight datasets are: Student data, Course, Department, Program, Faculty, Classroom, Timetable, Enrollment These tables are connected through appropriate relationships, and sample data has been inserted into each one to enable queries, reporting, and further experimentation.
